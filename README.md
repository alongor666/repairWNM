# 🐉 龙行健 · 全息健康修复 SOP

> 基于 2018-2026 全量医疗数据挖掘，针对 C2 型萎缩性胃炎与代谢超载的定制化修复白皮书

## 📋 项目概述

本项目是一个基于现代医学数据挖掘的健康修复标准化操作程序（SOP），专门针对以下健康问题：

- **C2 型萎缩性胃炎** - 胃粘膜防御修复
- **代谢超载** - 尿酸管理与肝脏代谢优化
- **心肺功能维护** - 预防性健康管理

## 🌐 在线访问

**🔗 官网地址**: [https://alongor666.github.io/repairWNM/](https://alongor666.github.io/repairWNM/)

## ✨ 核心功能

### 🎨 主题模式
- **🌙 暗色主题（默认）** - 护眼模式，防止刺眼
- **👁️ 老花眼模式** - 大字体模式，适合中老年用户

### 📊 健康资产审计
- **胃粘膜防御状态** - 实时风险评估
- **尿酸管理** - 代谢指标监控
- **肝脏代谢** - 功能状态评估
- **心肺储备** - 健康指标追踪

### 🍽️ 全息饮食矩阵
- **进取清单** - 推荐食物详细列表
- **绝对红线** - 禁忌食物明确标识
- **科学配比** - 基于医学数据的营养方案

### 📅 七日修复计划
- **分阶段修复** - 愈合期、修复期、固本期
- **精准配餐** - 早、中、晚、加餐全覆盖
- **个性化定制** - 基于个人健康数据

## 🛠️ 技术栈

- **前端框架**: HTML5 + Tailwind CSS
- **设计风格**: Apple风格设计语言
- **响应式**: 完美适配移动端和桌面端
- **部署平台**: GitHub Pages

## 🎯 设计理念

### 🌙 暗色主题优先
- 采用暗色主题作为默认模式
- 减少蓝光刺激，保护视力
- 适合长时间阅读和夜间使用

### 👁️ 老花眼友好
- 提供大字体模式切换
- 增强文字对比度
- 优化阅读体验

### 🍎 苹果设计美学
- 简洁优雅的界面设计
- 流畅的交互体验
- 现代化的视觉效果

## 📱 使用指南

1. **访问网站**: 点击上面的官网地址
